# Springboard Capstone Translation App



## About

- The translation app is built to provide a seamless user experience and interface where users can translate languages from any of their choice towards a target language. 
- Users have the ability to create accounts to store their translations and keep a record of their history. 


## Appeal
- The purpose of creating this application was to establish a seamless user experience where individuals were not clustered with other information on the website and can simply be provided the latest information.
